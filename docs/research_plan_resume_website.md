# 研究计划：简历改写网站技术调研和SEO策略

## 目标
- 评估AI模型，为简历改写平台提供技术选型建议
- 研究并制定简历改写的最佳实践和AI提示工程指南
- 分析SEO策略，提供针对性的优化方案
- 完成竞品分析，识别市场机会和差异化策略
- 汇总所有研究结果，形成最终的技术选择和策略建议文档

## 研究分解
[x] **第一部分：AI模型评估**
  - [x] 1.1 研究OpenAI GPT-4, Claude, Gemini等大语言模型的API文档
  - [x] 1.2 比较模型在简历改写任务上的性能和成本
  - [x] 1.3 撰写AI模型选择建议报告

[x] **第二部分：简历改写最佳实践**
  - [x] 2.1 研究专业简历改写的核心原则
  - [x] 2.2 探索AI提示工程在简历优化中的应用
  - [x] 2.3 收集并分析简历改写成功案例及模板
  - [x] 2.4 撰写简历改写AI提示工程指南

[x] **第三部分：SEO策略分析**
  - [x] 3.1 分析关键词 "resume editor", "linkedin", "简历优化" 的搜索趋势
  - [x] 3.2 研究竞品网站 (Zety, Resume.io, NovoResume) 的SEO策略
  - [x] 3.3 制定简历工具网站的SEO优化方案

[x] **第四部分：竞品分析**
  - [x] 4.1 分析现有工具的功能、特点和商业模式
  - [x] 4.2 研究付费功能的定价策略和转化方式
  - [x] 4.3 撰写竞品分析报告，识别市场机会

[x] **第五部分：最终报告**
  - [x] 5.1 整合所有研究成果
  - [x] 5.2 撰写最终的技术选型和策略建议文档

## 任务类型
- 主要焦点: 研究与分析
- 理由: 任务要求深入研究技术、市场和竞争对手，并基于研究结果提供策略建议。
